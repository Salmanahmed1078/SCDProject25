# Al-Dar Apparel Website Redesign

A modern, responsive website redesign for Al-Dar Apparel, focusing on UI/UX with clean placeholders for videos and images.

## Project Structure

```
/aldar-apparel/
├── index.html          # Landing page
├── products.html       # Products page
├── about.html          # About page
├── contact.html        # Contact page
├── css/
│   ├── styles.css      # Main styles and color variables
│   ├── components.css  # Component-specific styles
│   └── responsive.css  # Responsive design styles
├── js/
│   ├── main.js         # Main functionality
│   └── navigation.js   # Navigation handling
└── assets/
    ├── images/         # Image assets (for future use)
    └── videos/         # Video assets (for future use)
```

## Color Scheme

Based on the existing Al-Dar Apparel website:
- **Primary Colors**: Black (#000000), White (#FFFFFF)
- **Neutral Grays**: Light (#F5F5F5), Medium (#CCCCCC), Dark (#333333)
- **Accent Colors**: Blue (#4A90E2), Gold (#D4AF37)

## Features

### Landing Page
- Hero section with video placeholder (16:9 aspect ratio)
- "For Him" and "For Her" sections with image placeholders (16:9)
- "Why Al-Dar Apparel?" section with product detail and craftsmanship video placeholders

### Products Page
- Clean white background lookbook style (Farfetch-inspired)
- Product grid with image placeholders (1:1 aspect ratio)
- Product badges (BEST SELLER, NEW, TRENDING)
- Category navigation

### About Page
- Company story section
- Craftsmanship video placeholder (16:9)
- Workshop/showroom image placeholders

### Contact Page
- Contact form
- Contact information display
- Map placeholder

## Placeholders

All placeholders include:
- Clear labels indicating type (VIDEO PLACEHOLDER, IMAGE PLACEHOLDER)
- Aspect ratio information
- Content descriptions where applicable
- Proper aspect ratio maintenance using CSS padding-bottom technique

## Responsive Design

- Mobile-first approach
- Breakpoints: 768px (tablet), 1024px (desktop)
- Sections stack vertically on mobile
- Touch-friendly interactions

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Notes

This is a prototype focusing on UI/UX. All videos and images are placeholders with proper aspect ratios maintained. The design is inspired by:
- Prada.com for navigation and "For Him/For Her" layout
- Farfetch.com for clean white background lookbook style
- Giorgio Armani for craftsmanship video style reference
